PhD Student, Carnegie Mellon University

Co-founder of Qeexo

I specialize in building software that leverages data science, computer vision and machine learning to improve user interfaces. I have worked in the domains touch and free-space input , among others.

I have also developed a new user interface architecture which uses probabilities, not rules, to make decisions. This allows for the creation of more intelligent interfaces that can adapt to our behaviors, predict future actions, and use context to make better decisions.
